Inspired Homework (LTI 1.3 PoC)

A minimal Canvas-ready homework launcher for teachers and students.

Teacher view Student view

How it works

  1. Launch the tool from Canvas using LTI 1.3 deep linking.
  2. Create an assignment for individual students or a group.
  3. Post the generated launch link back to Canvas (simulated here).
  4. Students open the link to see their homework tasks.

This demo keeps assignments in memory so you can iterate quickly.
