Testing checklist

LTI-style
  1. Create an assignment in Teacher Studio and copy the student launch URL.
  2. Optional: simulate deep linking with the Canvas return URL field.
  3. Open the launch URL (or click Student View) to validate the learner flow.
  4. Use AI recommendations to populate tasks, then manually adjust before assigning to learners (No PII, learners grouped by Quartile performance).
